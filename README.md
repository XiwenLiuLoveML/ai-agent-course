# 🚀 30天成为智能体AI工程师（Zero to Hero）

👉 每天30分钟，从0开始，学会用AI做事

---

## 🎯 这门课在解决什么问题？

很多人开始接触AI的时候，会卡在这里：

- 会用ChatGPT，但不会做项目  
- 知道很多概念，但不知道怎么落地  
- 看了很多教程，但没有真正做出东西  

---

👉 这门课的目标很简单：

**让你在30天内，从“问AI问题”，变成“让AI帮你做事”。**

---

## 🧠 课程核心思路

我们不从复杂理论开始，而是从一个最直接的认知开始：

👉 **智能体 = 多次有结构的AI调用**

换句话说：

👉 不再只问一次  
👉 而是让AI一步一步完成任务  

---

## 📚 课程结构（每天30分钟）

每一天，你都会完成一个小任务：

| Day | 内容 | 产出 |
|-----|------|------|
| Day1 | 第一个智能体 | 需求判断助手 |
| Day2 | 什么是Agent | 理论框架 |
| Day3 | 多模型协作 | 小型AI智囊团 |
| ... | ... | ... |

👉 所有内容都围绕：

**理论 + 实践 + 可复用工具**

---

## 🧪 示例项目：需求判断智能体

👉 输入一句需求：

> 希望预测产品从研发到产线的一次合格率变化

👉 输出：

- 这是AI项目还是IT项目  
- 为什么  
- 可以怎么做  

---

👉 这个工具在真实企业（如新能源公司数智化中心）中可以直接使用

---

## 🛠 技术栈

- Python  
- Jupyter Notebook  
- 通义千问（Qwen API）  
- （后续）Gradio / 多Agent框架  

---

## 🎯 学完你能做到什么？

- 能写出自己的第一个AI智能体  
- 能把模糊需求拆解成结构化问题  
- 能设计简单的AI工作流  
- 能做出可以展示的AI项目（GitHub作品）  

---

## ❤️ 适合谁？

👉 完全0基础（不会写代码也可以）  
👉 想转AI工程师  
👉 想把AI用到真实工作中  

---

## 🚀 开始学习

👉 从 Day1 开始，直接运行 Notebook

---

## ✨ 一句话总结

👉 这不是一门教你“用AI”的课  
👉 而是一门教你“让AI替你工作”的课
